# Ajuste - Patrimônio e guia de assinatura

## Patrimônio

- Adicionado botão **Buscar** na página Patrimônio.
- O botão abre uma janela com até 10 campos para pesquisar 10 seriais/patrimônios de uma vez.
- Campos vazios são ignorados.
- Seriais repetidos na busca são ignorados para não duplicar resultados.
- Os resultados são consolidados em uma única lista.

## Guia de assinatura

- Adicionada a logo da Super Infra na guia de entrega/assinatura emitida pelo sistema.
- Atualizado o carimbo da guia para **CARIMBO DE CONFERÊNCIA SUPER INFRA**.

## Deploy

Alteração apenas de frontend. Após aplicar, executar build e publicar na Vercel.
